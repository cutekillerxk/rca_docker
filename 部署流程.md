问题根源
测试结果显示：
测试3b: localhost 禁用代理 → 成功返回 200
其他所有测试 → 返回 502
原因：Windows 系统可能通过注册表或系统设置自动配置了代理，导致 Python requests 库自动使用代理访问 localhost。代理无法正确处理本地请求，返回 502。
