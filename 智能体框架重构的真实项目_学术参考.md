### 2.1 统一数据集 + Role Token方案（推荐）

**论文支持：**

**《Prompt Tuning for Multi-Task Learning》** (Google, 2022)
- 核心：Prompt-based多任务学习
- 方法：Task-specific prompt + 统一模型
- 与你的方案：Role token = Task-specific prompt

**《Instruction Tuning with GPT-4》** (OpenAI, 2023)
- 核心：统一指令数据集微调
- 格式：`[Role] + Instruction + Input → Output`
- 与你的方案：完全一致

### 2.2 分离数据集的问题（学术证据）

**《Catastrophic Forgetting in Neural Networks》** (Kirkpatrick et al., 2017)
- 核心：灾难性遗忘问题
- 结论：连续微调会导致之前任务性能下降
- 与你的问题：分离数据集微调会相互干扰

**《Continual Learning in Neural Networks》** (Parisi et al., 2019)
- 核心：持续学习中的遗忘问题
- 解决方案：Experience Replay（统一数据集）
- 与你的方案：支持统一数据集策略
